# 👁️ Face Detection using OpenCV (Python)

A real-time face detection system built using Python and OpenCV. This project utilizes a pre-trained Haar Cascade classifier to identify human faces from a live webcam stream.

---

## 📌 Overview

This project demonstrates the implementation of classical computer vision techniques for face detection. It captures live video, processes each frame, and detects faces efficiently using OpenCV’s built-in models.

---

## ✨ Key Features

* Real-time face detection via webcam
* Lightweight and fast processing
* Pre-trained Haar Cascade classifier
* Simple and easy-to-understand implementation

---

## 🛠️ Technology Stack

* **Programming Language:** Python
* **Libraries:** OpenCV, NumPy
* **Model:** Haar Cascade Classifier

## 🧠 Working Principle

1. The webcam captures live video frames
2. Each frame is converted to grayscale
3. The Haar Cascade classifier detects faces
4. Bounding boxes are drawn around detected faces

---

## 📊 Output

The program opens a webcam window and highlights detected faces in real-time using rectangular bounding boxes.
